About TutorConnect

TutorConnect was inspired by my own struggles in finding the right tutor. I wanted to create a platform that makes it easier for students to connect with qualified tutors quickly and efficiently. This project is a Portfolio Project for Holberton School, showcasing my skills in Android development and user-centered design.
